Dynamic Functionality

The website will include three webpages with dynamic integration. The first will be a webpage that displays the top 20 stocks based on stock price. The user will be able to organize the top 20 stocks based on stock price, projected growth, or alphabetically. The stock prices will also update automatically based on current data from yahoo-fin's stock api. In addition to this, I will write a script that will put together, from the top 20 stocks, what it thinks the user should invest in right now, based on current growth. The third webpage will include a form that will allow the user to subscribe or hire a personal financial advisor. The user must be able to enter their payment information and create an account.
